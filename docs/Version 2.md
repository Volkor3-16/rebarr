# Oh lets just keep using shitty ass aggregators and scan sites forever!

I love needing to make endless number of connectors, providers, extensions, and other bullshit.

Why no Decentralised hosting? Store what you read, share what you read. 

IPFS + Orbit DB for metdata/syncing

## Cool stuff

- DMCA proof, the 'local it weeb' stores bulk manga, it automatically shares whats available with  other nodes
- Federation: Scanlators upload to any host, and anyone running a server can see it and store it
- Immutable w/ Versioning: Old versions still exist, but the best copy wins and gets stored by most
- Technically doesn't matter the content, could work for anything:
    - Hentai?
    - Random Comics?
    - Anime / TV / Movies????
